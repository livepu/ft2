### 2. 核心计算函数

| 函数名 | 参数 | 返回值 | 功能说明 |
|--------|------|--------|---------|
| `calculateCumulativeReturns` | `values[], baseIndex` | `number[]` | 以 baseIndex 为基准点计算累计收益率 |
| `calculateDrawdownFromNetValues` | `values[], baseIndex` | `number[]` | 基于净值计算回撤（百分比） |
| `getBenchmarkPrices` | `dates[], baseIndex` | `number[] \| null` | 获取基准价格序列，缺失值前值填充 |
| `calculateExcessReturns` | `strategyReturns[], benchmarkReturns[]` | `number[]` | 计算超额收益（策略 - 基准） |
| `calculateExcessDrawdown` | `excessReturns[]` | `number[]` | 计算超额收益的回撤 |

### 3. 核心算法流程（updateChartData）

```
步骤1: 计算可见区间索引
  startIdx = floor(length * startPercent / 100)
  endIdx = ceil(length * endPercent / 100) - 1
  baseIdx = startIdx > 0 ? startIdx - 1 : 0  // 前一交易日

步骤2: 提取可见区间数据
  visibleNetValues = fullData[baseIdx : endIdx+1]

步骤3: 计算策略收益
  returns = calculateCumulativeReturns(netValues, baseIdx)
  visibleReturns = returns[startIdx : endIdx+1]

步骤4: 计算策略回撤
  drawdownNetValues = fullData[startIdx : endIdx+1]
  drawdowns = calculateDrawdownFromNetValues(drawdownNetValues, 0)

步骤5: 获取基准数据
  benchmarkPrices = getBenchmarkPrices(dates, baseIdx)
  benchmarkReturns = calculateCumulativeReturns(...)
  benchmarkDrawdowns = calculateDrawdownFromNetValues(...)

步骤6: 计算超额收益/回撤
  excessReturns = calculateExcessReturns(visibleReturns, benchmarkReturns)
  excessDrawdowns = calculateExcessDrawdown(excessReturns)

步骤7: 计算区间统计
  periodReturn = visibleReturns[-1]
  relativeReturn = periodReturn - benchmarkPeriodReturn
  maxDrawdown = min(visibleDrawdowns)

步骤8: 数据对齐填充
  fullDrawdowns = [null, null, ...] + drawdowns
  fullBenchmarkReturns = [null, ...] + benchmarkReturns

步骤9: 更新图表
  myChart.setOption({ series, yAxis, legend })
```

## 四、关键差异点

### 1. 基准点机制

```javascript
// return2.html - 动态基准点
const baseIdx = startIdx > 0 ? startIdx - 1 : 0;  // 可见区间前一个交易日
const returns = calculateCumulativeReturns(netValues, baseIdx);

// report.js / analyzer.js - 固定基准点
const initial = this.assets[0];  // 永远是第一个数据点
```

**return2.html 的优势**：缩放时收益从0%开始，直观展示区间涨跌

### 2. 超额收益计算

```javascript
// return2.html - 独立计算超额收益和超额回撤
excessReturns = strategyReturns - benchmarkReturns
excessDrawdown = 基于超额收益的回撤（不减基准）

// report.js - 只有基准收益，无超额收益
```

### 3. 数据对齐

```javascript
// return2.html - 基准数据日期对齐 + 缺失值填充
benchmarkMap[date] = close;
if (!close) prices.push(prices[prices.length - 1]);  // 前值填充

// report.js - 简单过滤
.filter(item => date >= startDate && date <= endDate)
```

### 4. 命名规范对比

| 维度 | report.js | analyzer.js | return2.html |
|------|-----------|-------------|--------------|
| 方法命名 | `calculate_xxx` | `calc_xxx` | `calculateXxx` |
| 私有方法 | `_methodName` | `_methodName` | 无 |
| 参数命名 | `risk_free_rate` | `riskFreeRate` | `baseIndex` |
| 数据字段 | 中文 `日期`, `资产` | 英文 `date`, `assets` | 英文 `date`, `mkt_value` |

## 七、最终实现方案

### 确定的技术路径

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 代码组织 | 面向对象（类） | 数据管理清晰，支持链式调用 |
| 基准管理 | 函数接口传入 | 支持动态切换，触发重新计算 |
| 业绩基准日 | 区间前一交易日 | 与 return2.html 一致，缩放时收益从0%开始 |
| 输出方式 | 结构化数据 | ECharts 配置由外部构建，职责分离 |
| 方法命名 | `calc_` 前缀 | 简洁统一 |

### 类结构总览

```javascript
class AccountAnalyzer {
    // ========== 构造与属性 ==========
    constructor(dailyAssets)
    dailyAssets          // 净值数据
    benchmarkData        // 原始基准数据
    _alignedBenchmark    // 对齐后的基准数据

    // ========== 基准管理 ==========
    setBenchmark(benchmarkData)   // 设置基准，返回 this
    clearBenchmark()              // 清除基准，返回 this
    hasBenchmark()                // 是否有基准

    // ========== 时间区间 ==========
    getTimeRange(rangeType)       // 返回 {startIndex, endIndex, baseIndex, ...}

    // ========== 核心计算方法（calc_ 前缀）==========
    calcReturns(timeRange)        // 累计收益率序列
    calcDrawdowns(timeRange)      // 回撤序列
    calcExcess(timeRange)         // 超额收益序列
    calcReturnRate(timeRange)     // 区间收益率
    calcAnnualizedReturn(timeRange)  // 年化收益率
    calcVolatility(timeRange)     // 年化波动率
    calcMaxDrawdown(timeRange)    // 最大回撤
    calcSharpeRatio(timeRange, riskFreeRate)  // 夏普比率
    calcSortinoRatio(timeRange, riskFreeRate) // 索提诺比率
    calcVar(timeRange, confidence)   // VaR
    calcCvar(timeRange, confidence)  // CVaR
    calcUlcerIndex(timeRange)     // Ulcer Index
    calcUpi(timeRange, riskFreeRate) // UPI
    calcBenchmarkReturnRate(timeRange)  // 基准收益率
    calcBenchmarkMaxDrawdown(timeRange) // 基准最大回撤

    // ========== 输出方法 ==========
    getReturns(rangeType)         // 获取收益数据
    getDrawdowns(rangeType)       // 获取回撤数据
    getExcess(rangeType)          // 获取超额收益数据
    getStats(rangeType)           // 获取区间统计
    getResult(rangeType)          // 获取完整结果
}
```

### 使用示例

```javascript
// 初始化
const analyzer = new AccountAnalyzer(dailyAssets);

// 无基准获取结果
const result1 = analyzer.getResult('1y');

// 设置基准后获取结果

// 切换时间区间
const result3 = analyzer.getResult({start: '2024-01-01', end: '2024-06-30'});

// 清除基准
analyzer.clearBenchmark();
```
