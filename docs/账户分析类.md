# 账户分析类 (AccountAnalyzer)

## 设计思路

### 1. 数据初始化
- 传入 AccountManager 实例，自动聚合快照为日数据(_daily_assets)
- 计算交易盈亏记录(_trade_profits)
- 数据在初始化时固化，后续计算基于此数据

### 2. 时间区间切片
- `sliced_data`: 缓存当前区间的切片数据
- `getTimeRange(period)`: 设置时间区间，更新缓存
- `_ensure_sliced_data()`: 确保切片数据已初始化

### 3. 使用方式
- 默认使用全数据: 直接调用指标方法，自动初始化
- 指定区间: 先调用 `getTimeRange('3m')`，再调用指标方法
- 切换区间: 再次调用 `getTimeRange()` 即可

```python
>>> analyzer = AccountAnalyzer(account=my_account)
>>> analyzer.volatility()           # 使用全数据
>>> analyzer.getTimeRange('3m')     # 设置近3月区间
>>> analyzer.volatility()           # 使用近3月数据
>>> analyzer.sharpe_ratio()         # 复用缓存的近3月数据
```

### 4. 返回格式规范
- 所有比例类指标统一使用**小数**返回（如 0.15 = 15%）
- 显示时再乘以 100 转为百分比
- 天数类指标返回数值（如 5.2 天）

## 方法清单

### 一、资产类指标（支持时间切片）

| Python 方法 | JS 方法 | 中文名 | 返回类型 | 说明 |
|------------|---------|-------|---------|------|
| `return_rate()` | `calcReturnRate()` | 累计收益率 | `float` | 小数形式，如 0.15 表示 15% |
| `annualized_return()` | `calcAnnualizedReturn()` | 年化收益率 | `float` | 小数形式 |
| `volatility()` | `calcVolatility()` | 年化波动率 | `float` | 小数形式 |
| `sharpe_ratio()` | `calcSharpeRatio()` | 夏普比率 | `float` | 无单位 |
| `max_drawdown()` | `calcMaxDrawdown()` | 最大回撤 | `Tuple[float, date, date]` / `Object` | (回撤率, 起始日, 结束日) |
| `var(confidence=0.95)` | `calcVar()` | VaR | `float` | 小数形式 |
| `cvar(confidence=0.95)` | `calcCvar()` | CVaR | `float` | 小数形式 |
| `ulcer_index()` | `calcUlcerIndex()` | Ulcer Index | `float` | 小数形式 |
| `sortino_ratio(risk_free_rate=0.02)` | `calcSortinoRatio()` | 索提诺比率 | `float` | 无单位 |
| `upi(risk_free_rate=0.02)` | `calcUpi()` | UPI | `float` | 无单位 |

### 二、交易类指标（仅 Python，暂不支持时间切片）

| Python 方法 | 中文名 | 返回类型 | 说明 |
|------------|-------|---------|------|
| `win_rate()` | 胜率 | `float` | 小数形式，如 0.6 表示 60% |
| `avg_profit(mode='amount')` | 平均盈利 | `float` | mode: 'amount'金额, 'percentage'百分比 |
| `avg_loss(mode='amount')` | 平均亏损 | `float` | 负数 |
| `avg_profit_loss_ratio()` | 平均盈亏比 | `float` | 大于1为优 |
| `avg_holding_period()` | 平均持仓时间 | `float` | 天数 |
| `kelly_criterion()` | 凯利最优仓位 | `float` | 小数形式 |
| `kelly_fraction(fraction=0.5)` | 半凯利仓位 | `float` | 小数形式 |

### 三、序列计算方法（仅 JS，用于绘图）

| JS 方法 | 中文名 | 返回类型 | 说明 |
|---------|-------|---------|------|
| `calcReturns(timeRange)` | 收益率序列 | `Object` | `{dates, strategy, benchmark}` |
| `calcDrawdowns(timeRange)` | 回撤序列 | `Object` | `{strategy, benchmark, excess}` |
| `calcExcess(timeRange)` | 超额收益序列 | `Object` | `{returns, drawdowns}` |

### 四、基准管理方法（仅 JS）

| JS 方法 | 返回类型 | 说明 |
|---------|---------|------|
| `setBenchmark(benchmarkData)` | `this` | 设置基准数据 |
| `clearBenchmark()` | `this` | 清除基准数据 |
| `hasBenchmark()` | `boolean` | 是否有基准数据 |
| `calcBenchmarkReturnRate(timeRange)` | `number\|null` | 基准收益率 |
| `calcBenchmarkMaxDrawdown(timeRange)` | `Object\|null` | 基准最大回撤 |

### 五、区间控制方法

| Python 方法 | JS 方法 | 返回类型 | 说明 |
|------------|---------|---------|------|
| `getTimeRange(period_or_start, end)` | `getTimeRange(rangeType)` | `Dict` / `Object` | 获取/设置时间区间 |
| `_ensure_sliced_data()` | - | `Dict` | 确保切片数据已初始化（内部方法） |

**getTimeRange 参数说明：**
- Python: `period_or_start=None` 全数据, `'3m'` 近3月, `date(2024,1,1), end=date(2024,6,30)` 自定义
- JS: `'all'` 全数据, `'1m'/'3m'/'6m'/'1y'/'2y'/'ytd'` 区间, `{start, end}` 自定义

**返回格式：**
```python
{
    'startDate': date,      # 区间起始日期
    'endDate': date,        # 区间结束日期
    'baseDate': date,       # 基准日（计算收益率用）
    'daily_assets': Dict    # 切片后的日资产数据（仅 Python）
}
```

### 六、查询方法（仅 Python）

| Python 方法 | 返回类型 | 说明 |
|------------|---------|------|
| `daily_assets` | `Dict[date, float]` | 完整日资产数据 |
| `trade_profits` | `List[Dict]` | 完整交易盈亏记录 |
| `get_metrics()` | `Dict[str, Dict]` | 获取已计算指标 |
| `metrics()` | `Dict` | 收集所有 @metric 指标 |
| `returns(periods=None)` | `Dict` | 批量计算多区间收益率 |
| `get_daily_total_assets()` | `Dict[date, float]` | 同 daily_assets |
| `get_largest_profit_trades(n)` | `List[Dict]` | 盈利最大的N笔交易 |
| `get_largest_loss_trades(n)` | `List[Dict]` | 亏损最大的N笔交易 |

### 七、输出方法

| Python 方法 | JS 方法 | 返回类型 | 说明 |
|------------|---------|---------|------|
| `export_html(report_name, output_dir)` | - | `None` | 生成HTML报告 |
| - | `getReturns(rangeType)` | `Object` | 获取收益数据 |
| - | `getDrawdowns(rangeType)` | `Object` | 获取回撤数据 |
| - | `getExcess(rangeType)` | `Object\|null` | 获取超额收益数据 |
| - | `getStats(rangeType)` | `Object` | 获取区间统计汇总 |
| - | `getResult(rangeType)` | `Object` | 获取完整结果 |

**JS getStats 返回格式：**
```javascript
{
    startDate, endDate, baseDate,
    periodReturn, annualizedReturn, volatility,
    maxDrawdown, maxDrawdownStart, maxDrawdownEnd,
    sharpeRatio, sortinoRatio,
    var95, cvar95, ulcerIndex, upi,
    benchmarkReturn, benchmarkMaxDrawdown, relativeReturn
}
```

## 指标分类排序规范

| order范围 | 类别 | 指标 |
|----------|------|------|
| 1-9 | 基础信息 | 回测区间、初始资金、最终资产 |
| 10-19 | 收益指标 | 累计收益率、年化收益率 |
| 20-29 | 风险指标 | 年化波动率、最大回撤、VaR、CVaR、Ulcer Index |
| 30-39 | 风险调整收益指标 | 夏普比率、索提诺比率、UPI |
| 40-49 | 交易分析指标 | 胜率、平均盈亏比、平均持仓时间 |
| 50-59 | 仓位建议 | 凯利公式最优仓位、半凯利仓位 |
